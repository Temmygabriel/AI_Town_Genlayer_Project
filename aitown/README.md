# AI Town — GenLayer dApp

## Deploy in 5 minutes (no StackBlitz needed)

### Option A — Deploy via Vercel (easiest)

1. Download this project folder
2. Go to https://github.com/new and create a new repo
3. Upload / push these files to it
4. Go to https://vercel.com → New Project → Import your GitHub repo
5. Click Deploy — done!

---

### Option B — Deploy via terminal

```bash
# 1. Install dependencies
npm install

# 2. Test locally
npm run dev
# Open http://localhost:3000

# 3. Deploy to Vercel
npx vercel --prod
```
